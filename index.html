Mis trabajos

Landing Page

El primer proyecto corresponde a crear un Landing Page aplicando los conocimientos aprendidos con respecto a CSS. Para esto debemos definir un concepto y crearla. En mi caso se realizo con referencia a servicio y venta de productos para mascotas.

Iguana Page

Para este proyecto se busca poner a prueba nuestros conocimientos de la propiedad display y float, html semántico junto con todo lo aprendido previamente en HTML. Para esto debemos construir una pagina web utilizando una maqueta prestablecida y lograr llegar al mismo resultado.

Cuppon

Para este proyecto debemos aplicar todo lo conocimientos adquiridos con referencia al framework boostrap, construyendo una pagina para vender productos que deberá tener tanto una versión para escritorio, para Tablet y para teléfonos móviles.
